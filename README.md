# quick-links
Quick links help highlight the important pages on your website and with Bootstrap's column structure, you can easily create an efficient and stylish quick links section.

## Tutorial
For detailed instruction's, view Solodev's [How to Create Quick Links](http://www.solodev.com/blog/web-design/how-to-create-quick-links.stml)

## Demo
Try out a working example on [JSFiddle](https://jsfiddle.net/solodev/fx48z9r0/).
